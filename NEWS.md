# upsetly 0.1.0

* Prepared the first CRAN release.
* Added strict validation for data, column names, display limits, dimensions,
  and HTML target identifiers.
* Excluded elements that do not belong to any supplied set from intersection
  counts.
* Escaped user-provided labels in HTML tooltips and safely encoded `box_id` in
  the widget JavaScript.
* Reduced runtime dependencies and added automated tests.
